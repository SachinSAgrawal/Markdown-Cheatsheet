# Markdown Cheatsheet

## About
A cheatsheet for markdown document files and how to format text on them.

## Usage
Simply check out the [cheats](CHEATS.md) file!

* Lists
* Code
* Formatting
* Links
* Checklists
* Tables
* ... and more!
